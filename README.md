# portal-lgw-hub

## 项目简介

本仓库用于归档和托管多个独立的 HTML 页面。这些页面可以包含工具、信息展示、文档入口或其他静态内容，不针对特定网站或域名。仓库本身仅作为页面集合的发布与维护中心，不包含动态后端或数据库。

## 目录说明

```
portal-lgw-hub/
│
├── index.html          # 主入口页面（如页面索引或导航）
├── pages/              # 存放各独立 HTML 页面的文件夹
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 静态资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
├── README.md           # 本文件
└── LICENSE             # 许可证（如有）
```

- **pages/**：存放所有独立 HTML 页面，每个页面作为一个独立的归档单元。
- **assets/**：存放页面共享或专用的样式、脚本与图片资源。

## 页面归档说明

- 所有页面均为纯静态 HTML，不依赖外部服务或特定域名。
- 每个页面可独立访问，彼此之间无强制关联。
- 页面内容可能随维护更新而变化，但会尽量保持向后兼容。

## 使用方式

- 可直接通过 GitHub Pages 或其他静态托管服务部署本仓库内容。
- 将仓库克隆到本地后，使用任意 HTTP 服务器（如 Python 的 `http.server`）即可预览。
